### 3) MedSparseSNN｜医疗影像中的稀疏 SNN 隐私-效率研究（独立开发）
- 面向 DermaMNIST 与 PathMNIST，对比 SNN / DenseSNN / ANN 的准确率、成员推断风险与效率指标。
- PathMNIST：SNN test_acc=82.33±0.31，接近 ANN 的 85.12±0.40；同时 SNN 理论有效 MAC 节省 84.18±0.52%，优于 DenseSNN。
- DermaMNIST：SNN test_acc=69.93±0.20，理论有效 MAC 节省达到 90.74±0.23%，体现稀疏发放在计算节省上的稳定价值。
- 隐私侧：在当前黑盒 MIA 设定下，跨数据集完成 AUC/F1/precision/recall 评估，形成“隐私优势需结合数据域解释”的严谨结论。

技术栈：PyTorch、SpikingJelly、MedMNIST、MIA、实验报告自动化

### 4) PrivEnergyBench｜隐私-能效-效用三目标基准工具箱（独立开发）
- 抽象前述研究中的评测共性，开发统一 benchmark：训练、验证、MIA、时延测量、能耗代理、统计汇总、可视化输出。
- 支持 linear / MLP / CNN1D / Transformer 四类模型，多 seed 自动输出 mean/std/95% CI、Markdown 报告与 HTML dashboard。
- 默认 3-seed 结果：MLP 最高 val_acc=0.9208；Linear 最低能耗 0.0164 mJ/sample，形成清晰的性能-隐私-能耗权衡示例。
- 将“单次实验脚本”沉淀为“可复用评测基础设施”，提升实验公平性与复现效率。
